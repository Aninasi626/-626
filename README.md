# 📝 Vue2 记事本 (Todo App)

一个使用 **Vue 2** 构建的轻量级记事本/待办事项应用，适合前端初学者学习。

## ✨ 功能

- ➕ 添加待办事项
- ✅ 勾选标记完成（划线效果）
- ❌ 删除单个事项
- 🔍 按全部 / 未完成 / 已完成筛选
- 🧹 一键清除所有已完成事项
- 💾 自动保存到浏览器本地存储（刷新不丢数据）
- ⌨️ 支持回车键快捷添加

## 🚀 快速开始

直接双击 `todo-app.html` 用浏览器打开即可，无需安装任何依赖。

## 🧠 涵盖的 Vue 2 知识点

| 知识点 | 说明 |
|--------|------|
| `v-model` | 表单双向数据绑定 |
| `v-for` | 列表渲染 |
| `v-if` / `v-else` | 条件渲染 |
| `:class` | 动态类名绑定 |
| `@click` / `@keyup` | 事件处理 |
| `computed` | 计算属性 |
| `methods` | 方法定义 |
| `watch` | 深度监听数据变化 |
| `mounted` | 生命周期钩子 |
| `localStorage` | 数据持久化 |

## 📖 学习建议

适合刚学完 Vue 2 基础语法的同学练手，可以尝试这些改进：

1. 给事项添加**优先级**（高/中/低）
2. 添加**分类标签**（工作/生活/学习）
3. 用 `transition` 给列表添加**动画效果**
4. 拆分为**组件**（TodoInput / TodoList / TodoFilter）
5. 接入后端 API 实现**云端同步**
